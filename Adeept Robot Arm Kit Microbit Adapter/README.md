# Adeept Robot Arm Kit with Microbit
This kit is originally an Arduino Uno kit. To program it with micobit, it is connected with microbit on serial line and sending servo angle commands over serial.

![alt text](https://github.com/CoderDojo-Ennis/microbit/blob/main/images/adeept-microbit.png "adeept-microbit")
## User Manual
[Adeept arduino](https://www.adeept.com/learn/detail-64.html)

## Makecode Library
Please use this library to send serial servo commands from microbit.

To download the makecode library file, click on above file "MecanumCar_4WD_makecode_library.hex" and click download button as shown below
